一、大数据学习

Hadoop

Hadoop主要包含三大组件：HDFS+MapReduce+YARN

Hadoop的核心HDFS

HDFS（Hadoop Distributed File System）是一个分布式文件系统，用于存储海量数据。

1. 特点

  • 数据自动保存多个副本，且副本之间可自动备份
  • 数据自动切分，可按数据块存储到多个节点
  • 数据自动恢复
  • 数据高容错性

2. 组成

  • NameNode：负责管理文件元数据，包括文件路径、权限、副本数等
  • DataNode：负责管理本地文件存储，包括文件块的存储和恢复
  • SecondaryNameNode：辅助NameNode，分担其工作量
